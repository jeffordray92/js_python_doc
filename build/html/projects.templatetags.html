projects.templatetags module¶

The templatetags is a collection of extra functions that are built to provide specific functionalities in accessing Project objects.

projects.project_extras¶

projects.templatetags.project_extras.display_latest()[source]¶

Displays the five (5) latest created Projects, regardless of its type.

Projects displayed are ordered by its created and name attributes.

projects.templatetags.project_extras.display_latest_mobile()[source]¶

Displays the five (5) latest created mobile Projects, ordered by its created and name attributes.

projects.templatetags.project_extras.display_latest_web()[source]¶

Displays the five (5) latest created web Projects, ordered by its created and name attributes.
