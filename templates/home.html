Here's what JumpStart can do for you:


1. Create a new project and save its details in the database
Choose what type of project to ceate (mobile or web) Choose which template is to be applied to the web or mobile application created
2. Setup Project Repository
Create project repository on Ingenuity's Bitbucket account (GIT) Push initial commits on project repository (Commits depend on what type of project: iOS, Django and etc.)
3. Setup Project Management Tool (Redmine or JIRA)
Create new spaces on Redmine or Ingenuity's JIRA account, depending on which PM Tool the project requires.
4. Create a Test Site
Create project directory on Ingenuity's server. Create virtual environment for the project. Install packages or dependencies for the project. Create and assign sub-domain for the project. Create NGINX configuration file and restart server so test site would be up and running.
